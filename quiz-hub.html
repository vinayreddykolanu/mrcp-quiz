For team administrators

This is the front door — a launcher for three independent quiz applications (medical, nursing, pharmacy). Each quiz runs on GitHub Pages because most NHS SharePoint tenants block JavaScript in uploaded HTML files. The hub itself can live anywhere: a SharePoint document library, a Modern SharePoint Page (via the Embed or Quick Links web part), or directly on GitHub Pages alongside the three quiz files.

Update the URLs in the QUIZ_HUB_CONFIG.disciplines block at the top of the source file before deploying — relative paths if all four files share a folder, absolute URLs otherwise. To hide a discipline (e.g. you only run medical and nursing), set its enabled: false. No localStorage, no analytics, no tracking. Question rotation is fully deterministic — the same date returns the same question on every device, every year, in every discipline.
