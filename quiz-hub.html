For team administrators

This is the front door — a launcher for the daily question. The quiz itself runs on GitHub Pages because most NHS SharePoint tenants block JavaScript in uploaded HTML files. The hub can live anywhere: a SharePoint document library, a Modern SharePoint Page (via the Embed or Quick Links web part), or directly on GitHub Pages alongside the quiz.

Update the URLs in the QUIZ_HUB_CONFIG block at the top of the source file before deploying — relative paths if all files share a folder, absolute URLs otherwise. No localStorage, no analytics, no tracking. Question rotation is fully deterministic — the same date returns the same question on every device, every year.
