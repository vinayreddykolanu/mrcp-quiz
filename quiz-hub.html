For SharePoint administrators

This page is a single self-contained HTML file with no external CDN dependencies (other than Google Fonts, which most M365 tenants permit). It can be uploaded directly to a SharePoint document library, embedded inside a Modern SharePoint Page via the Embed web part, or linked to from any team site.

Update the three URLs in the QUIZ_HUB_CONFIG block at the top of the source file before deploying — paths can be relative if all files live in the same folder, or absolute SharePoint URLs if hosted across libraries. No localStorage, no analytics, no tracking — entirely offline once loaded.
